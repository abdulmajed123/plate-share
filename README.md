## Website Name

-- Food Share

## Website Live URL

-- https://foods-share.web.app/

## Features

--**User Authentication** using Firebase (Login, Register, Google Sign-In)
--**Add and Manage Food** — users can post surplus food items for donation
--**Request Food** — users can request available food and track request status
--**Responsive UI** built with TailwindCSS (Dark/Light mode support)

- **Admin Dashboard** for monitoring food donations and requests
  --**Protected Routes** — only logged-in users can access certain pages

---

## Technologies Used

--**Frontend:** React.js, TailwindCSS, React Router
--**Backend:** Node.js, Express.js, MongoDB
--**Authentication:** Firebase Auth
--**Hosting:** Firebase Hosting + Vercel (for API)
